# Typescript-Express-Mongo
*NodeJs boilerplate code using express and mongo*

## Running this app

1. Create an environment file from example
`cp .env.example .env`
2. Edit .env file with your mongodb server info
    1. DATABASE_HOST - *your mongodb hostmane*
    2. DATABASE_DB   - *your mongodb collection name*
    3. DATABASE_USER - *your mongodb server username*
    4. DATABASE_PASS - *your mongodb server password*
3. Install *node modules*
`npm install`
4. Start project
`npm start`
_Or_ Start project in watch mode
`npm run watch`
